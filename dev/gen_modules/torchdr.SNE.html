Implementation of the Stochastic Neighbor Embedding (SNE) algorithm introduced in [1].

Parameters:

  • perplexity (float) ‚Äì Number of ‚Äòeffective‚Äô nearest neighbors. Consider selecting a value between 2 and the number of samples. Different values can result in significantly different results.

  • n_components (int, optional) ‚Äì Dimension of the embedded space (corresponds to the number of features of Z).

  • lr (float, optional) ‚Äì Learning rate for the algorithm.

  • optimizer ({'SGD', 'Adam', 'NAdam'}, optional) ‚Äì Which pytorch optimizer to use.

  • optimizer_kwargs (dict, optional) ‚Äì Arguments for the optimizer.

  • scheduler ({'constant', 'linear'}, optional) ‚Äì Learning rate scheduler.

  • scheduler_kwargs (dict, optional) ‚Äì Arguments for the scheduler.

  • init ({'random', 'pca'} or torch.Tensor of shape (n_samples, output_dim), optional) ‚Äì Initialization for the embedding Z.

  • init_scaling (float, optional) ‚Äì Scaling factor for the initialization.

  • tol (float, optional) ‚Äì Precision threshold at which the algorithm stops.

  • max_iter (int, optional) ‚Äì Number of maximum iterations for the descent algorithm.

  • tolog (bool, optional) ‚Äì Whether to store intermediate results in a dictionary, by default False.

  • device (str, optional) ‚Äì Device to use, by default ‚Äúauto‚Äù.

  • keops (bool, optional) ‚Äì Whether to use KeOps, by default False.

  • verbose (bool, optional) ‚Äì Verbosity, by default True.

  • random_state (float, optional) ‚Äì Random seed for reproducibility, by default 0.

  • coeff_attraction (float, optional) ‚Äì Coefficient for the attraction term, by default 10.0 for early exaggeration.

  • coeff_repulsion (float, optional) ‚Äì Coefficient for the repulsion term, by default 1.0.

  • early_exaggeration_iter (int, optional) ‚Äì Number of iterations for early exaggeration, by default 250.

  • tol_affinity (float, optional) ‚Äì Precision threshold for the entropic affinity root search.

  • max_iter_affinity (int, optional) ‚Äì Number of maximum iterations for the entropic affinity root search.

  • metric_in ({'sqeuclidean', 'manhattan'}, optional) ‚Äì Metric to use for the input affinity, by default ‚Äòsqeuclidean‚Äô.

  • metric_out ({'sqeuclidean', 'manhattan'}, optional) ‚Äì Metric to use for the output computation, by default ‚Äòsqeuclidean‚Äô.
