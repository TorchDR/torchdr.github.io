class torchdr.affinity.L2SymmetricEntropicAffinity(perplexity: float = 30, tol: float = 1e-05, max_iter: int = 1000, sparsity: bool | None = None, metric: str = 'euclidean', device: str | None = None, keops: bool = False, verbose: bool = True)[source]ÔÉÅ

Bases: EntropicAffinity

Computes the L2-symmetrized entropic affinity matrix \(\overline{\mathbf{P}^{\mathrm{e}}}\) of t-SNE [2].

From the EntropicAffinity matrix \(\mathbf{P}^{\mathrm{e}}\), it is computed as

\[\overline{\mathbf{P}^{\mathrm{e}}} = \frac{\mathbf{P}^{\mathrm{e}} + (\mathbf{P}^{\mathrm{e}})^\top}{2} \:.\]
Parameters:

  • perplexity (float) ‚Äì Perplexity parameter, related to the number of ‚Äòeffective‚Äô nearest neighbors. Consider selecting a value between 2 and the number of samples.

  • tol (float, optional) ‚Äì Precision threshold at which the root finding algorithm stops.

  • max_iter (int, optional) ‚Äì Number of maximum iterations for the root finding algorithm.

  • sparsity (bool, optional) ‚Äì If True, keeps only the 3 * perplexity smallest element on each row of the ground cost matrix. Recommended if perplexity is small (<50).

  • metric (str, optional) ‚Äì Metric to use for computing distances, by default ‚Äúeuclidean‚Äù.

  • device (str, optional) ‚Äì Device to use for computation.

  • keops (bool, optional) ‚Äì Whether to use KeOps for computation.

  • verbose (bool, optional) ‚Äì Verbosity.

References
