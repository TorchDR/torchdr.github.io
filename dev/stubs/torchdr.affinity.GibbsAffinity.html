class torchdr.affinity.GibbsAffinity(sigma: float = 1.0, normalization_dim: int | Tuple[int] = (0, 1), metric: str = 'euclidean', nodiag: bool = True, device: str | None = None, keops: bool = True, verbose: bool = True)[source]ÔÉÅ

Bases: LogAffinity

Computes the Gibbs affinity matrix \(\exp( - \mathbf{C} / \sigma)\) where \(\mathbf{C}\) is the pairwise distance matrix and \(\sigma\) is the bandwidth parameter. The affinity can be normalized according to the specified normalization dimensions.

Parameters:

  • sigma (float, optional) ‚Äì Bandwidth parameter.

  • normalization_dim (int or Tuple[int], optional) ‚Äì Dimension along which to normalize the affinity matrix.

  • metric (str, optional) ‚Äì Metric to use for pairwise distances computation.

  • nodiag (bool, optional) ‚Äì Whether to set the diagonal of the affinity matrix to zero.

  • device (str, optional) ‚Äì Device to use for computations.

  • keops (bool, optional) ‚Äì Whether to use KeOps for computations.

  • verbose (bool, optional) ‚Äì Verbosity.
