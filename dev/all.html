Affinities torchdr.affinities

Simple affinities

GibbsAffinity([sigma, normalization_dim, ...])

Computes the Gibbs affinity matrix \(\exp( - \mathbf{C} / \sigma)\) where \(\mathbf{C}\) is the pairwise distance matrix and \(\sigma\) is the bandwidth parameter.

StudentAffinity([degrees_of_freedom, ...])

Computes the Student affinity matrix based on the Student-t distribution:

ScalarProductAffinity([normalization_dim, ...])

Computes the scalar product affinity matrix \(\mathbf{X} \mathbf{X}^\top\) where \(\mathbf{X}\) is the input data.

Entropic affinities

EntropicAffinity([perplexity, tol, ...])

Solves the directed entropic affinity problem introduced in [1].

L2SymmetricEntropicAffinity([perplexity, ...])

Computes the L2-symmetrized entropic affinity matrix \(\overline{\mathbf{P}^{\mathrm{e}}}\) of t-SNE [2].

SinkhornAffinity([eps, init_dual, tol, ...])

Computes the symmetric doubly stochastic affinity matrix \(\mathbf{P}^{\mathrm{ds}}\) with controlled global entropy using the symmetric Sinkhorn algorithm [5].

SymmetricEntropicAffinity([perplexity, lr, ...])

Computes the solution \(\mathbf{P}^{\mathrm{se}}\) to the symmetric entropic affinity problem (SEA) described in [3] with the dual ascent procedure.

Quadratic affinities

DoublyStochasticQuadraticAffinity([eps, ...])

Computes the symmetric doubly stochastic affinity matrix with controlled global \(\ell_2\) norm.
