Main module torchdr

Affinity module torchdr.affinity

EntropicAffinity([perplexity, tol, ...])

Solves the directed entropic affinity problem introduced in [R1333502c7ceb-1].

SymmetricEntropicAffinity([perplexity, lr, ...])

Computes the solution \(\mathbf{P}^{\mathrm{se}}\) to the symmetric entropic affinity problem (SEA) described in [R10aba7b01f7e-3] with the dual ascent procedure.

L2SymmetricEntropicAffinity([perplexity, ...])

Computes the L2-symmetrized entropic affinity matrix \(\overline{\mathbf{P}^{\mathrm{e}}}\) of t-SNE [Rc6132c17e3ca-2].

DoublyStochasticEntropic([eps, init_dual, ...])

Computes the symmetric doubly stochastic affinity matrix \(\mathbf{P}^{\mathrm{ds}}\) with controlled global entropy using the symmetric Sinkhorn algorithm [R3be52251fbbb-5].

false_position(f, n[, begin, end, max_iter, ...])
