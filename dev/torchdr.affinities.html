Affinities

Base structure and simple examples

All affinities inherit the structure of the Affinity() class.

torchdr.affinity.Affinity

Base class for affinity matrices.

If computations can be performed in log domain, the LogAffinity() class should be used.

torchdr.affinity.LogAffinity

Base class for affinity matrices in log space.

They can be used as a building block for DR algorithms.

>>> import torch

torchdr.affinity.GibbsAffinity([sigma, ...])

Computes the Gibbs affinity matrix \(\exp( - \mathbf{C} / \sigma)\) where \(\mathbf{C}\) is the pairwise distance matrix and \(\sigma\) is the bandwidth parameter.

Affinities based on entropic normalization

torchdr.affinity.EntropicAffinity([...])

Solves the directed entropic affinity problem introduced in [1]_.

torchdr.affinity.SymmetricEntropicAffinity([...])

Computes the solution \(\mathbf{P}^{\mathrm{se}}\) to the symmetric entropic affinity problem (SEA) described in [3]_ with the dual ascent procedure.

torchdr.affinity.L2SymmetricEntropicAffinity([...])

Computes the L2-symmetrized entropic affinity matrix \(\overline{\mathbf{P}^{\mathrm{e}}}\) of t-SNE [2]_.

torchdr.affinity.DoublyStochasticEntropic([...])

Computes the symmetric doubly stochastic affinity matrix \(\mathbf{P}^{\mathrm{ds}}\) with controlled global entropy using the symmetric Sinkhorn algorithm [5]_.

Other various affinities

torchdr.affinity.DoublyStochasticQuadratic([...])

Computes the symmetric doubly stochastic affinity matrix with controlled global \(\ell_2\) norm.
