DRModule

class torchdr.DRModule(n_components: int = 2, device: str = 'auto', keops: bool = False, verbose: bool = False, random_state: float = 0)[source]

Bases: TransformerMixin, BaseEstimator, ABC

Base class for DR methods. Each children class should implement the fit method.

Parameters:

  • n_components (int, default=2) – Number of components to project the input data onto.

  • device (str, default="auto") – Device on which the computations are performed.

  • keops (bool, default=False) – Whether to use KeOps for computations.

  • verbose (bool, default=False) – Whether to print information during the computations.

  • random_state (float, default=0) – Random seed for reproducibility.

abstract fit(X, y=None)[source]

Fits the DR model to project the input data \(\mathbf{X}\) onto a low-dimensional space.

This method must be overridden by subclasses. This base implementation only converts the input data \(\mathbf{X}\) to a torch tensor with the right device.

Parameters:

  • X (array-like object of shape (n_samples, n_features)) – or (n_samples, n_samples) if precomputed is True Input data or input affinity matrix if it is precomputed.

  • y (None) – Ignored.

Returns:

self – Fitted Estimator.

Return type:

object
