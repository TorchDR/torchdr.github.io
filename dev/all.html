API and Modules

Dimensionality Reduction Modules - sklearn Compatible

Base Dimensionality Reduction Module

DRModule([n_components, device, keops, ...])

Base class for DR methods.

Spectral Embedding

PCA([n_components, device, verbose, ...])

Affinity Matcher

AffinityMatcher(affinity_in, affinity_out[, ...])

Performs dimensionality reduction by matching two affinity matrices.

Neighbor Embedding

Base Modules

NeighborEmbedding(affinity_in, affinity_out)

Solves the neighbor embedding problem.

SparseNeighborEmbedding(affinity_in, ...[, ...])

Solves the neighbor embedding problem with a sparse input affinity matrix.

SampledNeighborEmbedding(affinity_in, ...[, ...])

Solves the neighbor embedding problem with a sparse input affinity matrix and a stochastic estimation of the repulsive term.

Classical Neighbor Embedding Methods

SNE([perplexity, n_components, lr, ...])

Implementation of the Stochastic Neighbor Embedding (SNE) algorithm introduced in [1].

TSNE([perplexity, n_components, lr, ...])

Implementation of the t-Stochastic Neighbor Embedding (t-SNE) algorithm introduced in [2].

TSNEkhorn([perplexity, n_components, lr, ...])

Implementation of the TSNEkhorn algorithm introduced in [3].

Noise Contrastive Neighbor Embedding Methods

InfoTSNE([perplexity, n_components, lr, ...])

Implementation of the InfoTSNE algorithm introduced in [15].

LargeVis([perplexity, n_components, lr, ...])

Implementation of the LargeVis algorithm introduced in [13].

UMAP([n_neighbors, n_components, min_dist, ...])

Implementation of the UMAP algorithm introduced in [8] and further studied in [12].

Affinity Modules

GaussianAffinity([sigma, metric, zero_diag, ...])

Computes the Gaussian affinity matrix \(\exp( - \mathbf{C} / \sigma)\) where \(\mathbf{C}\) is the pairwise distance matrix and \(\sigma\) is the bandwidth parameter.

StudentAffinity([degrees_of_freedom, ...])

Computes the Student affinity matrix based on the Student-t distribution.

ScalarProductAffinity([device, keops, verbose])

Computes the scalar product affinity matrix \(\mathbf{X} \mathbf{X}^\top\) where \(\mathbf{X}\) is the input data.

SelfTuningAffinity([K, normalization_dim, ...])

Computes the self-tuning [22] Gaussian affinity matrix with sample-wise bandwidth \(\mathbf{\sigma} \in \mathbb{R}^n\).

NormalizedGaussianAffinity([sigma, metric, ...])

Computes the Gaussian affinity matrix \(\exp( - \mathbf{C} / \sigma)\) where \(\mathbf{C}\) is the pairwise distance matrix and \(\sigma\) is the bandwidth parameter.

SinkhornAffinity([eps, tol, max_iter, ...])

Computes the symmetric doubly stochastic affinity matrix \(\mathbf{P}^{\mathrm{ds}}\) with controlled global entropy using the symmetric Sinkhorn algorithm [5].

EntropicAffinity([perplexity, tol, ...])

Solves the directed entropic affinity problem introduced in [1] and later used in TSNE [2].

SymmetricEntropicAffinity([perplexity, lr, ...])

Computes the solution \(\mathbf{P}^{\mathrm{se}}\) to the symmetric entropic affinity (SEA) problem described in [3].

DoublyStochasticQuadraticAffinity([eps, ...])

Computes the symmetric doubly stochastic affinity matrix with controlled global \(\ell_2\) norm.

UMAPAffinityIn([n_neighbors, tol, max_iter, ...])

Computes the input affinity used in UMAP [8].

UMAPAffinityOut([min_dist, spread, a, b, ...])

Computes the affinity used in embedding space in UMAP [8].

Utils

pairwise_distances(X[, Y, metric, keops])

Compute pairwise distances matrix between points in two datasets.

binary_search(f, n[, begin, end, max_iter, ...])

Performs a batched binary search to find the root of an increasing function f.

false_position(f, n[, begin, end, max_iter, ...])

Performs a batched false position method to find the root of an increasing function f.
