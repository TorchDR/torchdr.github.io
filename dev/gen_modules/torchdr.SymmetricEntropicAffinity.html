SymmetricEntropicAffinityÔÉÅ

class torchdr.SymmetricEntropicAffinity(perplexity: float = 30, lr: float = 1.0, eps_square: bool = True, tol: float = 0.001, max_iter: int = 500, optimizer: str = 'Adam', tolog: bool = False, metric: str = 'euclidean', nodiag: bool = True, device: str = 'auto', keops: bool = False, verbose: bool = True)[source]ÔÉÅ

Bases: LogAffinity

Computes the solution \(\mathbf{P}^{\mathrm{se}}\) to the symmetric entropic affinity problem (SEA) described in [3] with the dual ascent procedure. It amounts to the following convex optimization problem:

\[\begin{split}\mathbf{P}^{\mathrm{se}} \in \mathop{\arg\min}_{\mathbf{P} \in \mathbb{R}_+^{n \times n}} \: &\langle \mathbf{C}, \mathbf{P} \rangle \\ \text{s.t.} \quad &\mathbf{P} \mathbf{1} = \mathbf{1} \\ &\forall i, \: \mathrm{h}(\mathbf{P}_{i:}) \geq \log (\xi) + 1 \\ &\mathbf{P} = \mathbf{P}^\top \:.\end{split}\]

where :

  • \(\mathbf{C}\): symmetric pairwise distance matrix between the samples.

  • \(\xi\): perplexity parameter.

  • \(\mathrm{h}\): (row-wise) Shannon entropy such that \(\mathrm{h}(\mathbf{p}) = - \sum_{i} p_{i} (\log p_{i} - 1)\).

  • \(\mathbf{1} := (1,...,1)^\top\): all-ones vector.

It is a symmetric version of EntropicAffinity, where we simply added symmetry as a constraint in the optimization problem.

The algorithm computes the optimal dual variables \(\mathbf{\mu}^\star \in \mathbb{R}^n\) and \(\mathbf{\varepsilon}^\star \in \mathbb{R}^n_{>0}\) using dual ascent. The affinity matrix is then given by

\[\forall (i,j), \: P^{\mathrm{se}}_{ij} = \exp \left( \frac{\mu^\star_{i} + \mu^\star_j - 2 C_{ij}}{\varepsilon^\star_i + \varepsilon^\star_j} \right) \:.\]

Note

Unlike L2SymmetricEntropicAffinity that is used in t-SNE, SymmetricEntropicAffinity enables to control the entropy and mass of each row/column of the affinity matrix.
